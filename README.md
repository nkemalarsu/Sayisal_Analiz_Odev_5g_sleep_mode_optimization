# Sayisal_Analiz_Odev_5g_sleep_mode_optimization
5G ağlarında uyku modu süresinin Newton-Raphson yöntemi ile sayısal optimizasyonu.


# 5G BAZ İSTASYONLARINDA ENERJİ TASARRUFU İÇİN UYKU MODU SÜRE OPTİMİZASYONU (Newton-Raphson)

Bu proje, Sayısal Analiz dersi dönem projesi kapsamında geliştirilmiştir. 5G baz istasyonlarında enerji verimliliği ve hizmet kalitesi (QoS) arasındaki dengeyi optimize eden matematiksel bir modelin sayısal çözümünü elde etmek için tasarlanmıştır..

## Proje Amacı
5G ağlarında enerji tasarrufu sağlamak amacıyla baz istasyonunun uyku moduna geçmesi gerekmektedir. Ancak uyku süresi uzadıkça gecikme cezası üstel olarak artar. Bu proje, Toplam Maliyet Fonksiyonunu minimize eden optimum uyku süresini bulmayı hedefler.

## Kullanılan Yöntem
Problemin çözümü için **Newton-Raphson Sayısal Analiz Yöntemi** kullanılmıştır.
- **Dil:** Python 3.x
- **Kütüphaneler:** `numpy`, `matplotlib`, `math`
